О проекте

Почему Electronic Diary?

Мы создаём единый цифровой кабинет, в котором школа может вести дневники, контролировать успеваемость и держать связь с родителями.

Миссия
Цифровизация образования

Electronic Diary — это современная платформа управления школой, которая объединяет всех участников образовательного процесса в одной системе. Мы убираем бумажную волокиту и делаем коммуникацию прозрачной.

🎯

Единая система

Администрирование, расписание и оценивание находятся в одном приложении без лишних интеграций.

👁

Прозрачность

Родители видят прогресс ребёнка в реальном времени, а учителя получают удобные инструменты.

📈

Масштабирование

Подходит как для небольшой школы, так и для сети образовательных учреждений.

🔒

Безопасность

JWT-аутентификация, ролевая модель доступа и шифрование паролей по стандарту BCrypt.

⚡

Быстрый старт

Зарегистрируйте школу, получите пароль и начните работу за несколько минут.

🌐

REST API

Полноценный REST API с документацией Swagger для интеграции с другими системами.

🛠 Технологический стек

Java 17 Spring Boot 3 Spring Security Spring Data JPA PostgreSQL Flyway JWT Swagger / OpenAPI Docker Lombok MapStruct
